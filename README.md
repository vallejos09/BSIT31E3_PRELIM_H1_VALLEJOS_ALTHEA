Student Management System – Procedural Core

Objective
This project is a console-based Student Management System built in C#.  
It demonstrates fundamental programming concepts such as variables, loops, conditionals, methods, and data structures.

Requirements
- C# Console Application
- Menu-driven interface
- Program runs continuously until the user selects Exit
- Data stored in memory using List or Array
- System must compute:
  - Individual student averages
  - Class average
  - Highest grade and top student

---

Menu Options
1. Add Student  
2. Display Students  
3. Individual Averages  
4. Class Average  
5. Top Student  
6. Exit  

Usage Instructions
1. Run the program in a C# console environment.  
2. Choose an option from the menu by entering the corresponding number.  
3. Follow input prompts (e.g., enter student name and grades).  
4. View results such as averages or top student.  
5. Continue until you select Exit.

Expected Behavior
- Adding a student stores their name and grades in memory.  
- Displaying students shows all records with grades.  
- Individual averages are computed per student.  
- Class average is computed across all students.  
- Top student is identified based on the highest average.  
- Program continues until Exit is chosen.
